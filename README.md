# Responsive Warehouse

[![CI](https://github.com/suxls/responsive-warehouse/actions/workflows/ci.yml/badge.svg)](https://github.com/suxls/responsive-warehouse/actions/workflows/ci.yml)

Latency-aware multi-agent reinforcement learning for autonomous warehouse
coordination. A from-scratch study in optimization under real-time
responsiveness constraints, built as my last independent project before
graduating.

> Status: **Phase 0 — Foundations.** See [`plan.md`](plan.md) for the full
> roadmap and [`journal/`](journal/) for design decisions as they happen.
